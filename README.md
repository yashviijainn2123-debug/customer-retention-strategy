# 📊 Customer Retention Strategy Dashboard

## 🧠 Overview
Customer retention is a key factor in business success. This project presents an interactive **Customer Retention Strategy Dashboard** built using **Streamlit**, which analyzes customer behavior, identifies churn patterns, and provides actionable insights.

The dashboard combines key performance indicators, data visualization, and segmentation techniques to support data-driven decision-making.

---

## 🚀 Features

- 📌 **Key Performance Indicators (KPIs)**
  - Churn Rate
  - Average Balance
  - Active Users %
  - Average Products per Customer

- 📈 **Engagement vs Churn Analysis**
  - Compares active vs inactive customers
  - Shows impact of engagement on churn

- 🛍️ **Product Utilization Analysis**
  - Relationship between number of products and retention
  - Demonstrates importance of cross-selling

- ⚠️ **High-Value Disengaged Customers**
  - Identifies customers with high balances but low activity
  - Highlights high-risk churn segments

- 📊 **Customer Segmentation**
  - High Value Active
  - Moderate
  - High Value At Risk

- 📉 **Balance Distribution**
  - Visualizes customer balance patterns using histogram

## 🔍 Key Insights

- Active users have lower churn rates  
- More products lead to better retention  
- High-balance inactive customers are high risk  
- Segmentation improves targeted strategies  

---

## 💡 Recommendations

- Target high-value inactive customers  
- Promote multi-product usage  
- Improve engagement strategies  
- Monitor low-activity users
